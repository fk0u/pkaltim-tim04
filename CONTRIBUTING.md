# Contributing to BorneoTrip

Terima kasih atas minat Anda untuk berkontribusi pada BorneoTrip! Dokumen ini berisi panduan untuk membantu Anda memulai.

## Tata Cara Kontribusi

### 1. Pelaporan Bug (Issues)
Jika Anda menemukan bug, silakan buat Issue baru dengan format:
- **Judul**: Deskripsi singkat bug.
- **Langkah Reproduksi**: Cara untuk memunculkan bug tersebut.
- **Ekspektasi**: Apa yang seharusnya terjadi.
- **Screenshot**: (Opsional) Bukti visual.

### 2. Pengajuan Fitur
Kami terbuka untuk ide baru! Buat issue dengan label `enhancement` dan jelaskan user value dari fitur tersebut.

### 3. Pull Requests (PR)
1.  **Fork** repository ini.
2.  Buat **Branch** baru untuk fitur Anda (`git checkout -b fitur-keren`).
3.  **Commit** perubahan Anda (`git commit -m 'Menambahkan fitur keren'`).
4.  **Push** ke branch (`git push origin fitur-keren`).
5.  Buat **Pull Request** ke branch `main` repository ini.

## Coding Standards

- **TypeScript**: Gunakan tipe data yang eksplisit (hindari `any` jika memungkinkan).
- **Tailwind**: Gunakan class utility standar. Hindari style inline.
- **Komponen**: Pecah komponen besar menjadi komponen yang lebih kecil dan reusable.
- **Naming**: PascalCase untuk Component (`PackageCard.tsx`), camelCase untuk fungsi/variabel (`handleBooking`).
